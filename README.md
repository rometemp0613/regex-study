### Part 1: 정규표현식 입문 (Ch.1)

  - 핵심: 정규식 정의, 패턴 매칭, 검색/치환 개념
- [ ] **2. 정규표현식 작성 도구와 테스트 환경** (1.3)
  - 핵심: regex101, grep, sed, 온라인 도구

### Part 2: 기본 문법 (Ch.2 전반)

- [ ] **3. 리터럴 텍스트와 비인쇄 문자** (2.1~2.2)
  - 핵심: 문자 그대로 매칭, \n \t \r 등
- [ ] **4. 문자 클래스와 와일드카드** (2.3~2.4)
  - 핵심: [...], [^...], . (dot), 범위 지정
- [ ] **5. 앵커: 행 시작/끝, 단어 경계** (2.5~2.6)
  - 핵심: ^, $, \b, \B, 멀티라인
- [ ] **6. 유니코드와 다자택일** (2.7~2.8)
  - 핵심: \p{}, 유니코드 속성, | (파이프)
- [ ] **7. 그룹과 캡처** (2.9~2.11)
  - 핵심: (), 역참조 \1, 명명 캡처 (?P<name>)
- [ ] **8. 반복(수량자)** (2.12~2.13)
  - 핵심: *, +, ?, {n,m}, 탐욕적/게으른 수량자
- [ ] **9. 역행 제어와 성능** (2.14~2.15)
  - 핵심: 역행(backtracking), 소유적 수량자, 원자 그룹
- [ ] **10. 전후방탐색과 조건문** (2.16~2.17)
  - 핵심: lookahead (?=), lookbehind (?<=), 조건부 매칭

### Part 3: 치환과 주석 (Ch.2 후반)

- [ ] **11. 정규식 주석과 가독성** (2.18)
  - 핵심: (?#...), x 플래그, 자기문서화 정규식
- [ ] **12. 치환 텍스트 다루기** (2.19~2.22)
  - 핵심: $1, $&, $`, $', 치환 패턴

### Part 4: 프로그래밍 활용 (Ch.3)

- [ ] **13. 정규식 스타일과 라이브러리** (3.1~3.3)
  - 핵심: PCRE, POSIX, Python re, JS RegExp
- [ ] **14. 정규식 객체와 옵션** (3.4~3.5)
  - 핵심: 컴파일, re.IGNORECASE, re.MULTILINE 등
- [ ] **15. 매칭: 검사와 텍스트 추출** (3.6~3.10)
  - 핵심: match(), search(), group(), span()
- [ ] **16. 반복 매칭과 유효검사** (3.11~3.14)
  - 핵심: findall(), finditer(), 중첩 검색
- [ ] **17. 치환 기법** (3.15~3.19)
  - 핵심: sub(), 콜백 치환, 조건부 치환
- [ ] **18. 문자열 분리와 행 단위 처리** (3.20~3.22)
  - 핵심: split(), 행 단위 검색, 경계 유지 분리

### Part 5: 실전 레시피 — 유효검사 (Ch.4)

- [ ] **19. 이메일/전화번호 유효검사** (4.1~4.3)
  - 핵심: 이메일 패턴, 북미/국제 전화번호
- [ ] **20. 날짜/시간 유효검사** (4.4~4.7)
  - 핵심: 구식 날짜, ISO 8601, 시간 형식
- [ ] **21. 입력 제한과 텍스트 검증** (4.8~4.11)
  - 핵심: 영숫자 제한, 길이, 행 수, 긍정 응답
- [ ] **22. 신분/식별번호 검증** (4.12~4.14)
  - 핵심: SSN, 주민등록번호, ISBN
- [ ] **23. 주소와 우편번호** (4.15~4.18)
  - 핵심: 미국/캐나다/영국 우편번호, 사서함
- [ ] **24. 이름 형식 변환과 카드번호** (4.19~4.21)
  - 핵심: 이름↔성 변환, 신용카드, VAT 번호

### Part 6: 실전 레시피 — 텍스트 처리 (Ch.5)

- [ ] **25. 단어 검색 기초** (5.1~5.3)
  - 핵심: 정확한 단어 매칭, 복수 단어, 유사 단어
- [ ] **26. 단어 제외/부정 검색** (5.4~5.6)
  - 핵심: 특정 단어 제외, 부정 전후방탐색
- [ ] **27. 근접/중복 단어 처리** (5.7~5.8)
  - 핵심: 단어 간 거리, 중복 단어 감지
- [ ] **28. 행 단위 처리** (5.9~5.11)
  - 핵심: 중복 행 제거, 특정 단어 포함/제외 행
- [ ] **29. 공백 처리와 이스케이프** (5.12~5.14)
  - 핵심: 트림, 연속 공백 압축, 메타문자 이스케이프

- [ ] **33. URL 유효검사와 검색** (7.1~7.5)
